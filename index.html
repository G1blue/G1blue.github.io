How to use?

For the general calculator, the operation symbol you need to use for each operation is: + (for addition), - (for subtraction), * (for multiplication), and / (for division). For the other calculators, input the data in the first textbox before inputting into the second one. If you want to change the values, change the first input, then the second input.
