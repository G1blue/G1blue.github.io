About:
These calculators about to be shown are calculators that I made myself. I developed an algorithm so the calculators would work properly.

How to use the general calculator?

  1. Type the largest number in the first box.
  2. Type the smallest number in the second box.
  3. Choose your operation.
  4. Click "Calculate".

How to use the many addends calculator?

  1. Type the smallest number in the first box.
  2. Type the largest number in the second box.
  3. Click "Calculate".

How to use the root calculator?

  1. Type the root (square root = 2, cube root = 3, etc.)
  2. Type a number.
  3. Click "Calculate".

How to use the exponent calculator?

  1. Type the exponent.
  2. Type a number.
  3. Click "Calculate".
